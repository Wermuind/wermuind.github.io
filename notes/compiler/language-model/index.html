Модель языка

Символ/знак (symbol) — объект, которому соответствует какой-то определённый смысл.
Алфавит (alphabet) — конечное множество уникальных символов.
Строка (цепочка, слово; string) — любая конечная последовательность символов алфавита.
Язык — подмножество всех строк в алфавите.

Способы задания языка:

  • конечный язык: если язык содержит конечное число строк, то его можно задать простым перечислением этих строк
  • метод распознавания: бесконечный язык можно задать алгоритмом (процедура, которая всегда заканчивается), который определяет, принадлежит ли строка языку
  • метод порождения: язык можно задать порождающей грамматикой

Грамматикой называется набор 4 объектов:

  1. N — множество нетерминальных символов
  2. T — множество терминальных символов (T и N не пересекаются)
  3. P — конечное множество правил (продукций)
  4. S — начальный знак (аксиома) грамматики

Терминальный символ (terminal symbol) — объект, непосредственно присутствующий в словах языка, соответствующего грамматике, и имеющий конкретное символьное значение.

Нетерминальный символ (non-terminal symbol) — объект, обозначающий какую-либо сущность языка, но не имеющий конкретного символьного значения.

Правило грамматики (production rule) — правило перезаписи, определяющее замену символов, которая может выполняться рекурсивно для создания новых последовательностей символов.

Контекстно-свободная грамматика​

Классификация грамматик (Иерархия Хомского):

  • Тип 0 — неограниченная
  • Тип 1 — контекстно-зависимая
  • Тип 2 — контекстно-свободная
  • Тип 3 — регулярная

Контекстно-свободная грамматика (КС-грамматика, context-free grammar, CFG) — формальная грамматика, правила которой могут применяться к нетерминальному символу независимо от его контекста. Язык, порождённый КС-грамматикой называется контекстно-свободным языком.

Пример: a b(); в языке C это объявление функции, а в языке C++ — объявление функции или инициализация переменной в зависимости от контекста.

Каждое правило КС-грамматики записывается в виде A → β, где где A — один нетерминальный символ, а β — строка терминалов и/или нетерминалов.
